Приклад оформлення таблиці за допомогою CSS

Що тут використано з CSS

  • Рамки для таблиці та комірок (`border`)
  • Об'єднання рамок (`border-collapse: collapse`)
  • Внутрішні відступи комірок (`padding`)
  • Фон таблиці та комірок (`background-color`)
  • Фіксована висота заголовків (`height` для `th`)
  • Вирівнювання тексту по горизонталі (`text-align`)
  • Вирівнювання по вертикалі (`vertical-align`)
  • Чергуються рядки («зебра») через псевдоклас `:nth-child(even)`
  • Зміна фону рядка при наведенні (`tr:hover`)
  • Зміна відстані між рамкою таблиці і вмістом сторінки (`margin`)
