Мої роботи з курсу «Основи WEB-дизайну»

На цій сторінці зібрані всі виконані завдання. Оберіть роботу, щоб перейти до окремого прикладу.

Список робіт

Усі сторінки знаходяться в одному репозиторії GitHub у різних папках. Нижче розміщені кнопки-посилання, які відкривають потрібне завдання в новій вкладці.

1. Таблиці за допомогою CSS

Приклад оформлення таблиць: рамки, відступи, фон, «зебра»-рядки та зміна фону при наведенні.

Відкрити роботу з таблицями

2. Заголовки і великий обсяг тексту

Структура довгого тексту із заголовками різних рівнів, форматуванням абзаців, зображеннями та внутрішніми посиланнями.

Відкрити роботу із заголовками

3. Посилання та кнопки за допомогою CSS

Документ з великим текстом, посиланнями, кількома видами оформлених кнопок та кнопкою з JavaScript, що змінює фон сторінки.

Відкрити роботу з посиланнями та кнопками

Як використовувати цю сторінку

Ця сторінка може бути головною для GitHub Pages. Посилання на неї матиме вигляд https://ім'я-користувача.github.io/ім'я-репозиторію/web-links/. Звідси вже можна переходити до окремих завдань у папках headings, tables та links.
